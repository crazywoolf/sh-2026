# Этап 7 — конкурентные находки

Цель: `https://team-004.aisouthhack.ru`

## 1. Робастность (никогда не 500)

- пустое тело → 400: HTTP **400** 
- кривой JSON → 400: HTTP **400** 
- нет поля вопроса → 400/422: HTTP **422** 
- длинный вход (6000+): HTTP **200**
- prompt injection: HTTP **200**
- SQL-инъекция: HTTP **200**
