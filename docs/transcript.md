## 2. Extractor — извлечение данных
- **под-вопрос:** Какое отношение LTV/CAC по сегментам? Где привлечение убыточно?
- **approach:** metric_template (метрика ltv_cac_by_segment)
- **data_sufficient:** true · **строк:** 3
- **SQL (исполнен на DuckDB read-only):**
```sql
SELECT segment, round(avg(ltv_12m)/avg(cac),2) AS ltv_cac
          FROM unit_economics_monthly WHERE month='2025-12-01' GROUP BY 1 ORDER BY 2 DESC
```
